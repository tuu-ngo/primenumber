Prime Number Checker

Enter any integer to find out if it's a prime number

Accepts positive and negative integers

API Reference

Use the REST API to integrate prime checking into your application.

GET /api/check/{number}
POST /api/check 
curl -X POST \
  /api/check \
  -H "Content-Type: application/json" \
  -d '{"number": 17}'

Example Response

{
  "isPrime": true,
  "number": 17,
  "error": null,
  "message": "17 is a prime number."
}
